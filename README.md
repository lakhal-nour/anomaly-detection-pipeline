# Anomaly Detection Pipeline — DevSecOps + MLOps

Pipeline de détection d anomalies dans les logs réseau bancaires
avec apprentissage non supervise (Isolation Forest) et architecture DevSecOps complete.

## Stack technique
- ML : Isolation Forest / Autoencoder
- MLOps : MLflow
- API : FastAPI
- Streaming : Apache Kafka
- CI/CD : GitHub Actions
- Securite : Bandit, Safety, Trivy, OWASP ZAP
- Monitoring : Prometheus + Grafana
- Dataset : CICIDS2017

## Dataset
data/raw/cicids2017_cleaned.csv — 685 Mo
